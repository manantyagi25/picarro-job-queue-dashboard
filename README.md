# Picarro Job Queue Dashboard

This is a Next.js + TypeScript project scaffolded manually for building a job queue dashboard.

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

   Then open `http://localhost:3000` in your browser.

## Scripts

- `npm run dev` – start the development server.
- `npm run build` – build the production bundle.
- `npm run start` – start the production server.
- `npm run lint` – run ESLint.

## Project structure

- `src/app/layout.tsx` – root layout and global configuration.
- `src/app/page.tsx` – main landing page.
- `src/app/globals.css` – base styles and minimal utility classes.

You can now extend this scaffold with your job queue views, API routes, and any additional configuration you need.
